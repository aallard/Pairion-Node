# Sounds

This directory holds prerendered sound-design samples shipped with the Node.

In M0, no sound files are present. At M1/M6, this will contain:
- `wake-chime.opus` — played on wake-word detection
- `acknowledgment-tone.opus` — brief acknowledgment cue
- `thinking-thrum.opus` — ambient processing sound
- `completion-sigh.opus` — turn completion cue
- `offline-transition.opus` — descending tone on offline transition
- `online-restore.opus` — rising tone on reconnect
- `error-cached-*.opus` — cached error messages for Dumb-tier offline mode

Samples are curated from permissively-licensed sources or commissioned.
